Privacy Policy

Last updated: March 28, 2026

The short version

Facsimile does not collect, store, or transmit any personal data. No analytics, no tracking, no accounts.

What Facsimile does

Facsimile is a free, open-source Chrome extension that captures the visual structure of a web page and copies it to your clipboard so you can paste it into Figma as editable layers.

Data collection

Facsimile collects no data whatsoever. Specifically:

  • No personal information is collected
  • No browsing history is recorded
  • No analytics or telemetry is sent
  • No cookies are set
  • No accounts or sign-ups are required

Network requests

The extension makes a single network request to mcp.figma.com to fetch Figma's open-source page capture script. This request contains no user data — it simply downloads a JavaScript file. No other network requests are made by the extension.

Permissions

Facsimile requests two Chrome permissions:

  • activeTab — allows the extension to interact with the page you're currently viewing, only when you click the extension icon
  • scripting — allows the extension to inject Figma's capture script into the current page

These permissions are the minimum required for the extension to function. The extension cannot access your tabs in the background or read pages you haven't explicitly chosen to capture.

Data storage

Facsimile does not store any data locally or remotely. The captured page content is placed on your clipboard and is not retained by the extension after capture.

Third parties

Facsimile does not share data with any third parties because it does not collect any data. The capture script fetched from Figma is governed by Figma's own privacy policy.

Children's privacy

Facsimile does not collect any data from any users, including children under 13.

Changes to this policy

If this policy changes, the updated version will be posted on this page with a revised date. Since Facsimile collects no data, meaningful changes are unlikely.

Contact

Questions about this policy can be directed to the GitHub repository.
